# Dashboard — Streamlit

Web dashboard with 7 pages. Dark theme, auto-refresh 60s.

## Scripts
- `app.py` — Streamlit main with sidebar navigation
- `page_market_posture.py` — Dalio 4-quadrant, exposure gauge, posture card
- `page_portfolio.py` — Allocation donut, positions table, equity curve, heatmap
- `page_analysis.py` — Candlestick + overlays, RSI/MACD/OBV, radar chart, fundamentals
- `page_screener.py` — Sortable table, filters, drill-down, AI recommended badge
- `page_macro.py` — GDP/CPI/yield curve/VIX charts, regime timeline, Polymarket odds
- `page_paper_trading.py` — Equity curve, trade log, risk metrics, win/loss distribution
- `page_settings.py` — Config UI for portfolio, alerts, API keys, risk parameters

## Theme
- Dark background
- Green accent: #00D26A
- Red accent: #FF4444
- Auto-refresh: 60 seconds
